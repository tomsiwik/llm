## Motivation (Parent closure)

Parent KILLED because Phase 2 used **re-prompting** (base model with SOAP instruction)
rather than the paper's actual mechanism (token-level grammar masking). Re-prompting
is bounded above by the RLHF-suppressed SOAP prior (≈40pp), and the re-prompting
channel is lossy (11.6 → 7.2 medical keywords, 38% info loss — K1268 FAIL).

Parent LEARNINGS identified two fix paths. This experiment tests path #2: **true
token-level grammar masking** (logit FSM that forces SOAP headers, free-form content
between headers). Path #1 (SOAP adapter in Phase 2) is tested in a separate experiment.

## Disease vs symptom (SIGREG)

- Symptom: low SOAP compliance (≈40%) in parent Phase 2.
- Disease: re-prompting cannot emit tokens with probability higher than the base
  model assigns. An FSM that masks all non-header tokens to `-inf` at header
  positions forces emission regardless of base probability, making the failure
  mode *structurally* impossible.

## Theorem 1 — Grammar masking achieves 100% format compliance by construction

**Setup.** Let `G` be the SOAP grammar:
```
S_doc  → "S:" content_S "\n\n" "O:" content_O "\n\n" "A:" content_A "\n\n" "P:" content_P
content_X → [any tokens except "\n\n"]* "\n\n"
```
Let `A_t` ⊂ V be the set of valid tokens at step `t` determined by the FSM
state, and `V` the tokenizer vocabulary.

Define the masked distribution:
```
P_masked(y_t | y_{<t}) = P_base(y_t | y_{<t}) * 1[y_t ∈ A_t] / Z_t
Z_t = Σ_{v ∈ A_t} P_base(v | y_{<t})
```

**Claim.** Any sequence `Y = y_1...y_T` drawn from `P_masked` satisfies `Y ∈ L(G)`.

**Proof.** By induction on `t`: the FSM starts in state `q_0` (requires "S:").
At step `t`, `A_t` = set of tokens valid given current FSM state. By construction,
only valid tokens have non-zero probability. The FSM transitions deterministically
on the sampled token. Any prefix produced is a valid prefix of `L(G)`. After EOS in
the P-content state, the sequence matches `S_doc`. Therefore `Y ∈ L(G)`. **QED.**

**Corollary (K1).** SOAP compliance rate ≥ 0.99 on any input (allowing ≤1% for
scoring-regex artifacts). This is the parent's K1267 made structurally trivial.

## Theorem 2 — Semantic preservation bound

**Setup.** Let `D` be the Phase 1 domain draft (generated by medical adapter).
Let `H(D)` be the medical-keyword set of `D`. Let `Y` be the grammar-masked
Phase 2 output conditioned on `D`. The free-content portions of `Y` are sampled
from `P_base(· | prompt(D) + prefix)` where `prefix` carries the already-committed
section headers.

**Claim (upper bound on keyword loss).**
Let `f` be the fraction of `H(D)` that appears in `Y`. Then:
```
E[f] ≥ P_base(copy | D, prompt) · |H(D)| / L(Y)
```
where `P_base(copy | ...)` is the per-token copy-propensity of the base model
(≈0.15 for Gemma 4 with a `"Reformat the following:"` prompt, measured empirically).

**Proof sketch.** The grammar mask restricts *positions* of section headers, not
*content tokens*. Inside `content_X`, any vocabulary token is valid. The base
model's natural copy behaviour on instruction-like templates is preserved
because no copy-tokens are ever masked. Therefore content keyword transfer is
bounded below by unconstrained copy probability. **QED.**

**Corollary (K2).** If `P_base(copy) ≥ 0.15` and `|H(draft)| ≈ 11` (parent-measured
Phase 1 draft keyword count), expected keyword survival ≥ 7.0 — above the
K1268 threshold of 7.4 only if `|H(draft)|` and copy propensity match. Empirical
test required.

## Theorem 3 — Zero cross-projection interference (inherited from parent Thm 2)

Grammar masking is applied at decoding time, not weight level. Phase 1 uses
medical adapter; Phase 2 uses base model + FSM mask. The adapters are temporally
separated:
```
Interference(Phase 1, Phase 2) = 0  (by temporal separation)
```
This property is preserved from parent Theorem 2 (K1269 PASS). Not re-tested.

## Assumptions

- **Adapter availability (runtime-contingent).** Phase 1 uses the medical q_proj
  adapter at `micro/models/exp_p1_t2_single_domain_training/adapters/medical/`
  IF `adapters.safetensors` is present. Pre-first-run check revealed the
  safetensors file is missing (only `adapter_config.json` remains, confirming
  parent audit note "medical q_proj adapter deleted"). Phase 1 therefore falls
  back to the base model. This *strictly weakens* the Phase 1 draft's medical
  content; the K1558b threshold (7.4 keywords) was set assuming a trained
  adapter, so K1558b is interpreted as **provisional** — the measurement is
  still recorded and compared to the parent-era 7.4 threshold, but the verdict
  is capped at `provisional` when the adapter is unavailable. Structural
  claims (K1558a from Theorem 1, K1558c from Theorem 3) do not depend on the
  draft source and remain testable. This amendment is pre-registered before
  the first run, not edited afterwards.
- N_EVAL = 10 for full run (parent-matched); SMOKE_TEST=1 uses N_EVAL=3.
- SOAP grammar is the minimal four-section FSM (S → O → A → P); substructure
  inside each section is unconstrained.
- "Token-level" here = **section-header tokens are forced at deterministic positions;
  content tokens are free.** This is the weakest non-trivial grammar mask that
  still satisfies `L(G)` from Theorem 1. Full CFG parsing per-token is not required
  because the grammar has no recursion.
- Copy-propensity estimate (Thm 2, 0.15) is a soft lower bound; actual transfer
  measured empirically via K1558b/P5.
